## Overview

Development of a warehouse management system app using Flutter with GetX state management, collaborating with cross team members for a capstone project during MSIB 5.

## Key Features

- **Warehouse Rental**: Rent a warehouse according to preference Pay the rent fee with midtrans.
- **Verify Identity With ML**: Verivy identity when user register into the app, machine learning retrieving strings on cards through the application.
- **Find Warehouse on Maps**: User can find warehouse easily with interactive maps and can add the warehouses into their favorites.
- **Manae Warehouse**: User can manage the warehouse kinda like price, location, image, description and etc.
- **AI Chatbot**: Ask some question to bot powered by OpenAI.

## Technologies Used

- **Figma**: For creating and managing design assets and prototypes.
- **Flutter**: For android mobile development.
- **Firebase**: For login authentication and cloud data storage.
- **GetX**: For state management.
- **Open AI API**: For Chatbot features.
